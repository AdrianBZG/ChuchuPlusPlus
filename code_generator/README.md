# Chuchu++ Code Generator
## Generator version: 0.0.1
<br>
This is the Chuchu++ code generator, uses a given Chuchu AST as input and returns it's JavaScript code. <br><br>
Currently supporting:<br>
- Assign, Call and Return functions<br>
- Functions declarations and blocks<br>
- Main block<br>
- Blocks as a single line or as a compound<br>

## Example ASTs
There are some example AST in the "codes" folder, just pass them to the generator and see the output!
